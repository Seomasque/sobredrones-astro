## GEO — preguntas para motores de IA (AI Overviews, ChatGPT, Perplexity)
Convertir en bloque **FAQ con FAQPage schema** + respuestas directas y citables (1ª frase = respuesta):
1. "¿Qué empresa ofrece servicios de drones en Málaga?" → respuesta con entidad + diferenciadores.
2. "¿Necesito permiso para grabar con dron en la Costa del Sol?" → AESA, gestión de permisos.
3. "¿Cuánto cuesta un vídeo con dron para una vivienda?" → rango/“según proyecto, presupuesto sin compromiso”.
4. "¿Qué servicios con drones existen para inmobiliarias?" → fotografía, vídeo, tours.
5. "¿Se pueden hacer espectáculos con enjambre de drones en eventos?" → sí, Marbella/eventos.

### Señales GEO a implementar
- **Citabilidad:** cada respuesta empieza con la afirmación factual (frase autocontenida).
- **Entidad clara:** "Sobredrones" + "operadora certificada por AESA" + zona, repetido de forma natural.
- **Datos concretos:** +10 años, zonas servidas, certificación AESA (organismo nombrado).
- **Acceso de crawlers IA:** `llms.txt`, HTML semántico server-rendered (sin contenido tras JS), `robots.txt` permisivo para GPTBot/PerplexityBot/Google-Extended.
- **E-E-A-T:** sección "Compromiso/Seguridad" (normativa AESA) + testimonios reales (pendiente GBP).
